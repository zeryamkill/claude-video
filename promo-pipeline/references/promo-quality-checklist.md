# Promo Video Quality Checklist

Run through this before exporting any promo video.

## Visual

- [ ] Color consistency: no jarring temperature shifts between adjacent scenes
- [ ] Text readability at 1080p: headline 56px+ minimum, subtext 24px+
- [ ] No clipped whites or crushed blacks in stock footage
- [ ] Ken Burns direction varies (not all zoom-in)
- [ ] Transitions match scene energy (fast scenes get short transitions)
- [ ] CTA scene has visual emphasis (larger text, stronger backing)

## Audio

- [ ] Voiceover at -12 LUFS (clear above music)
- [ ] Music at -18 LUFS (sits under VO with 6 LU separation)
- [ ] SFX at -14 LUFS (audible but not dominant)
- [ ] No clipping: true peak below -1 dBTP
- [ ] Music ducks smoothly during voiceover (no hard volume jumps)
- [ ] Ducking varies by scene intent (CTA ducks harder than proof)
- [ ] Music fades in over 1s+ at start, fades out over 2s+ at end

## Text

- [ ] Minimum 2 seconds display per phrase (3s base + 0.5s per word)
- [ ] Text appears 0.3s before voiceover starts (visual priming)
- [ ] Safe zone: 200px top, 100px bottom, 10% sides
- [ ] Contrast backing activates on bright/busy backgrounds
- [ ] No text overlapping scene transition boundaries

## Pacing

- [ ] Hook scene: 3-5 seconds (no longer)
- [ ] CTA scene: 3-4 seconds (urgent, not lingering)
- [ ] Total under 60 seconds for social, under 90 for YouTube
- [ ] Energy arc: Hook (high) > dip > build > CTA (peak)
- [ ] Pattern interrupt every 3-5 seconds (new scene, text, or SFX)

## Platform

- [ ] Aspect ratio matches target (16:9 landscape, 9:16 vertical)
- [ ] Codec: H.264 for universal compatibility
- [ ] Pixel format: yuv420p
- [ ] -movflags +faststart applied
- [ ] File size within platform limits (287MB TikTok, 250MB Reels, 200MB LinkedIn)

## Flow

- [ ] No jarring color jumps between scenes (use longer fade if needed)
- [ ] Transitions reinforce narrative (cut for urgency, fade for softness)
- [ ] SFX vary per transition (not same sound repeated)
- [ ] Scene intents follow logical arc (hook > problem > feature > cta)
